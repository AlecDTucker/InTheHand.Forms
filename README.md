
# InTheHand.Forms
Extra Xamarin Forms APIs

Supports Android, iOS and Windows platforms.

[![NuGet version](https://badge.fury.io/nu/inthehand.forms.svg)](https://badge.fury.io/nu/inthehand.forms)

## Contains:-
- CheckSwitch
- ColorExtensions (UWP)
- DesignMode
- MediaElement
- NavigationPage2

## Platforms
- Android
- iOS
- Windows UWP
- Windows 8.1 Store
- Windows Phone 8.1
